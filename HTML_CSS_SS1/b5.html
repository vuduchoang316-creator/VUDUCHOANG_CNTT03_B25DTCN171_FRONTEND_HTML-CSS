Giới thiệu về Môn Frontend

Frontend là gì?

Frontend là phần quan trọng của phát triển web, bao gồm mọi thứ mà người dùng có thể nhìn thấy và tương tác trực tiếp trên trang web

Các công nghệ Frontend phổ biến

  • HTML (HyperText Markup Language): Ngôn ngữ đánh dấu siêu văn bản, dùng để tạo cấu trúc cơ bản của trang web.
  • CSS (Cascading Style Sheets): Ngôn ngữ định kiểu, dùng để thiết kế giao diện và bố cục của trang web.
  • JavaScript: Ngôn ngữ lập trình giúp làm cho trang web trở lên động, với các tính năng như xử lý sự kiện , hiệu ứng động, và kết nối với máy chủ.

Vai trò của Frontend

Frontend giúp tạo ra một trải nghiệm người dùng mượt mà và dễ dàng tiếp cận. Những nhà phát triển frontend đảm bảo trang web tương thích trên các thiết bị và trình duyệt khác nhau.
